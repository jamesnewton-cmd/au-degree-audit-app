Professor / Advisor Scheduling

Upload class listings + student requests (Excel) and generate student schedules.

Class listing workbook should include at least: Course Code, and optionally Section, Course Title, Instructor, Capacity, Days, Start Time, End Time or Meeting Time.
Student requests workbook should include Student ID, Student Name, and either Requested Courses or columns like Request 1, Request 2, etc.
Input Files
One row per section. Times are used to prevent class-time conflicts for each student.
Request order is treated as priority during scheduling.
